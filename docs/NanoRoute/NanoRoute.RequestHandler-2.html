Delegate RequestHandler<TRequest, TResponse>

Represents a request handler in the router pipeline.

Namespace: NanoRoute
Assembly: NanoRoute.dll
Syntax
public delegate TResponse RequestHandler<TRequest, TResponse>(RequestContext<TRequest> requestContext, Func<TResponse> callNext)
Parameters
Type Name Description
RequestContext<TRequest> requestContext

The current request context, including parsed route parameters and services.
Func<TResponse> callNext

Invokes the next compatible handler in the pipeline.
Returns
Type Description
TResponse

The response produced by the current handler, or by a later handler when callNext is invoked.
Type Parameters
Name Description
TRequest

The application-specific request type.
TResponse

The application-specific response type.
Examples
router.AddHandler("GET", "/api/users/{user_id:int}/", (requestContext, callNext) =>
{
    requestContext.Parameters["User"] = LoadUser((int) requestContext.Parameters["user_id"]!);
    return callNext();
});
