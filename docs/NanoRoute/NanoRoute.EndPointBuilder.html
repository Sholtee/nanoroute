public sealed class EndpointBuilder
Remarks

Endpoint builders capture the endpoint route pattern and HTTP verbs once, then let endpoint-aware extensions register middleware against that captured endpoint. Extension authors can use Prefix when they need access to the endpoint's scoped route builder.

Examples
builder.AddEndpoint("POST", "/users/", endpoint => endpoint
    .WithJsonBody<CreateUserRequest>("body")
    .WithHandler((context, _) => CreateUser(context)));
