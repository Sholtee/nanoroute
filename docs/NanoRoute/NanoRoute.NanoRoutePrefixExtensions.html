Class NanoRoutePrefixExtensions

Adds prefix-routing helpers that configure child route builders under a shared base pattern.

Remarks

Prefix builders inherit the value parsers and metadata visible when the child builder is created. Later changes made inside the prefix scope stay local to that child branch.

AddPrefix<TBuilder>(TBuilder, string, Action<RouteBuilder>)

Creates a child builder for the given prefix, invokes a configuration callback, and returns the current builder.
