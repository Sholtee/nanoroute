Delegate ParameterParserDelegate

Tries to parse a single route segment into a value that can be stored in Parameters.

Namespace: NanoRoute
Assembly: NanoRoute.dll
Syntax
public delegate bool ParameterParserDelegate(string segment, out object? parsed)
Parameters
Type Name Description
string segment

The raw path segment extracted from the request URI.
object parsed

The parsed value when the delegate returns true; otherwise null.
Returns
Type Description
bool

true when the segment is accepted by the parser; otherwise false.
Examples
router.AddParameterParser("int", (string segment, out object? parsed) =>
{
    if (int.TryParse(segment, out int value))
    {
        parsed = value;
        return true;
    }

    parsed = null;
    return false;
});
